## Solution in javascript

```js
/**
 * @return {null|boolean|number|string|Array|Object}
 */
Array.prototype.last = function () {
  let lastItemIndex = this.length - 1;

  if (this.length === 0) {
    return -1;
  }
  return this[lastItemIndex];
};

/**
 * const arr = [1, 2, 3];
 * arr.last(); // 3
 */
```
